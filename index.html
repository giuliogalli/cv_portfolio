XYZ - Laboratorio di Ricerca e Co-progettazione, La Scuola Open Source
July 2018
An open source and generative typeface designed during the last edition of XYZ, a cooperative design workshop held in Bari (Italy) by La Scuola Open Source, as the corporate font for Officina degli Esordi, one of the most important cultural containers of southern Italy. All the typefaces are generated by a special notational and programming system. First, a text file is designed to depict the glyph, then, a python script scan the file and applies basic shapes (randomly) to it.
